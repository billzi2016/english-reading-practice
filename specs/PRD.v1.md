# 英语阅读练习网站 PRD

> 版本说明：本文档记录的是 v1 静态 HTML/CSS/JS 版本的产品需求，已归档到 Astro v2 项目的 `specs/` 目录中，便于后续追溯旧版本目标和迁移范围。

## 项目概述

这是一个个人英语阅读练习网站，通过阅读真实的人物传记、技术介绍、历史故事等内容来提升英语阅读能力。

## 目标用户

- 有一定英语基础，希望通过阅读提升阅读理解能力的学习者

## 功能需求

### 文章页面（0001_xxx.html, 0002_xxx.html ...）
- 每篇文章约 500~800 词
- 内容真实，涵盖：人物传记、技术介绍、历史事件
- 难词/专业词汇在文中高亮，悬停显示中文翻译
- 阅读进度条
- 预计阅读时间显示
- 文章导航（上一篇 / 下一篇）
- 返回首页按钮

### 首页（index.html）
- 所有文章的列表/卡片展示
- 每张卡片显示：标题、分类标签、难度、摘要、阅读时间
- 分类筛选（人物传记 / 技术 / 历史）

### 技术选型
- HTML5 语义化标签
- Tailwind CSS（via CDN）用于样式
- 原生 JavaScript 用于交互
- 分离 CSS 和 JS 文件（custom.css, main.js）

## 设计风格

- 深色主题（深蓝/灰黑背景）
- 清晰的衬线字体用于正文，提升阅读体验
- 简洁卡片式布局
- 难词使用下划线虚线 + 高亮颜色标注

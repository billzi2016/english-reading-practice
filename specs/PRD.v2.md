# 英语阅读练习网站 PRD v2

> 本文档记录当前 Astro v2 项目的产品范围、技术结构和维护规则。  
> 重要约定：PRD 是产品与架构基线文档，不是文章内容清单。以后新增、删除、调整文章时，只修改 `src/data/articles.ts`，不要因为文章数量变化去修改 PRD。

## 项目概述

这是一个个人英语阅读练习网站，通过真实人物传记、技术介绍、历史故事等内容帮助学习者提升英语阅读理解能力。

v2 版本已经从 v1 的静态 HTML/CSS/JS 文件重构为 Astro 项目。页面结构、文章数据、交互逻辑和类型定义分开维护，减少重复代码，降低后续新增文章和改版成本。

## 目标用户

- 有一定英语基础，希望通过真实短文提升阅读理解能力的学习者
- 希望边读英文边快速理解难词中文含义的学习者

## 核心维护规则

- `src/data/articles.ts` 是文章数据的唯一来源。
- 新增文章只追加 `Article` 对象，不新增手写 HTML 页面。
- 上一篇 / 下一篇导航由 `src/utils/articles.ts` 根据文章数组顺序自动派生。
- 分类和难度展示配置只维护在 `src/config/`；分类体系可扩展，不限于 v1 的人物/技术/历史三类。
- 站内链接必须通过 `src/utils/routes.ts` 生成，兼容 GitHub Pages 子路径。
- `src/pages/articles/[slug].astro` 统一渲染所有文章详情页。
- `src/components/` 只放可复用 UI 组件。
- `src/scripts/interactions.ts` 只放浏览器交互逻辑。
- `src/types/index.ts` 维护文章领域类型，避免字段定义散落。
- PRD 不跟随文章数量变化更新；只有产品范围、架构、技术选型或维护规则变化时，才更新 PRD。

## 功能范围

### 首页

- 展示文章卡片列表
- 支持分类筛选
- 支持文本搜索
- 展示文章标题、摘要、分类、难度和预计阅读时间

### 文章详情页

- 展示文章标题、副标题和元信息
- 展示正文内容
- 高亮难词并显示中文解释
- 展示关键词汇表
- 展示阅读进度条
- 展示估算阅读时间
- 支持上一篇 / 下一篇导航
- 支持返回首页

### 主题与阅读体验

- 支持深色 / 浅色主题切换
- 默认深色主题
- 使用衬线字体优化长文阅读体验
- 使用统一的 `.vocab` 样式标注重点词汇

## SOLID / DRY 约束

- 单一职责：页面负责组合，组件负责展示，脚本负责交互，数据文件负责内容。
- 开闭原则：新增文章不改页面模板；新增分类时通过类型和映射表集中补齐。
- 接口隔离：首页卡片只依赖 `ArticleMeta`，不依赖完整正文。
- 依赖倒置：页面依赖文章数据和类型接口，不依赖旧静态 HTML 文件。
- DRY：所有文章共用动态路由模板，避免每篇文章维护一份重复页面结构。

## 启动与构建

```sh
# 在项目根目录执行
pnpm install
pnpm dev
```

```sh
pnpm build
pnpm preview
```

## 不做的事情

- 不在 PRD 中维护每篇文章的完整清单。
- 不为每篇文章新增独立 HTML 文件。
- 不把浏览器交互逻辑写散到多个页面里。
- 不把临时构建产物作为源码维护。
